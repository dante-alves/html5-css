Principais formatações

Negrito / Destaque / Força

Para deixar o texto em negrito, só preciso utilizar a tag <strong> que o texto ficará em negrito.

Itálico / Ênfase

Já para deixar o texto em itálico, eu utilizo a tag <em> que o texto ficará em itálico.

Marca texto

E para deixar o texto grifado, eu utilizo a tag <mark> que o texto ficará grifado :D

Texto grande e pequeno

Utilizava-se a tag <big> para deixar o texto grande, mas não se usa mais. Já o <small> não se tornou obsoleto, portanto pode ser utilizado e vai deixar o texto pequeno

Texto riscado e Texto sublinhado

Para riscar o texto, eu utilizo a tag <del>, enquanto para sublinar um texto, eu utilizo a tag <ins>

Texto sobrescrito e Texto subscrito

Para fazer coisas do tipo 20x5 utiliza-se a tag <sup>. Já para fazer coisas do tipo H2O, utiliza-se a tag <sub>
