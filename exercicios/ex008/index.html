Código-fonte e Pré formatação

Por exemplo no código document.getElementByID('teste'), que é escrito em linguagem JavaScript, a leitura dele fica ruim pois as lestras não estão mono-espaçadas. Portanto, eu posso utilizar a tag <code> para deixá-las mono-espaçadas, ficando assim: document.getElementByID('teste').

A tag code também pode ser utilizada diretamente sem precisar estar em um parágrafo:

num = int(input('Digite um número')) if num % 2 == 0> print(f'O número {num} é PAR') else: print(f'O número {num} é IMPAR') print ('Fim do programa')

No entanto, o código ficou todo numa linha, para deixar ele no estilo de código, utiliza-se a tag <pre>. Ex: 

        
            num = int(input('Digite um número'))

Citações simples


Para fazer uma citação simples, utiliza-se a tag <q>. Ex: 

Como diria meu pai: Cagar é uma ciência profunda, quando a bosta bate na água, a água bate na bunda.


Citações completas


Já para fazer uma citação completa, utiliza-se a tag <blockquote>. Ex: 
 

    O Naruto pode ser um pouco duro às vezes

    Ficar zangado ou se dar por vencido
    Ele está sempre disposto a melhorar

Eu também utilizei o código <blockquote cite=URL> para indicar de onde veio a citação.
